class Granularity(value)¶

Bases: Enum

Enum representing possible values of time granularity that coinbase API can handle. Values assigned to particular options represent number of seconds that certain option denotes.

FIFTEEN_MINUTES = 900¶
FIVE_MINUTES = 300¶
ONE_DAY = 86400¶
ONE_HOUR = 3600¶
ONE_MINUTE = 60¶
SIX_HOURS = 21600¶
THIRTY_MINUTES = 1800¶
